需求：
1、查询已激活的，并且博客的名字是包含某个查询字符串的记录
2、如果用户没有输入任何查询字符串，那么就显示所有已激活的博客

分析：
-- 如果用户输入了查询字符串
SELECT * FROM blog
WHERE state = 'ACTIVE'
AND title LIKE '%o%'

-- 如果用户没有输入任何查询字符串
SELECT * FROM blog
WHERE state = 'ACTIVE'


mapper：
注意：
1、参数类型如果是多值对象，如Blog类型，那么if语句中的test属性判断使用Blog的属性名
2、参数类型如果是简单类型，如String类型，那么if语句中的test属性判断使用value的属性名
如果使用简单类型（string），那么test中无法判断 空字符串""
如果使用$ 那么 传值应该使用 ${value}， if中判断也应该使用 value 关键字
如果使用# 那么 传值应该使用 ${任意名}， if中判断还是使用 value 关键字

因此：最好使用对象传参

    <!--*************** 动态sql ***************-->
    <select id="selectActiveBlogByTitle"  parameterType="Blog" resultMap="blogResultMap">
        SELECT * FROM blog 
        WHERE state = 'ACTIVE'
        <!-- <if test="title != ''">
            AND title LIKE '%${title}%'
        </if> -->
        <if test="title != null and title != ''">
            AND title LIKE #{title}
        </if>
    </select>

接口：
List<Blog> selectActiveBlogByTitle(Blog blog);
